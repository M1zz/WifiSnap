WifiSnap이란?

WifiSnap은 와이파이 공유와 연결을 간편하게 만들어 주는 iOS 앱입니다.

  • 내 와이파이 공유 — 지금 연결된 와이파이를 QR로 만들어, 친구가 카메라로 비추면 바로 접속합니다.
  • 촬영 → 연결 — 카페·숙소의 와이파이 안내문을 찍으면 ID/PW를 자동으로 읽어 연결합니다.
  • 근처 와이파이 추천 — 연결했던 장소에 다시 오면 그 와이파이를 목록 맨 위로 올려줍니다.
