개인정보 처리방침

최종 업데이트: 2026년 7월 21일

요약

WifiSnap은 어떤 개인정보도 수집·전송·판매하지 않습니다. 모든 데이터는 사용자의 기기 안에서만 처리되고 저장됩니다. 별도의 서버가 없으며, 광고·분석·추적 도구를 사용하지 않습니다.

앱이 사용하는 권한과 데이터

카메라 · 사진

와이파이 안내문에서 ID/비밀번호를 인식(OCR)하기 위해 사용합니다. 이미지 분석은 전적으로 기기 내에서(on-device) 이뤄지며, 사진이나 인식 결과가 외부로 전송되지 않습니다.

위치

연결한 와이파이의 위치를 기억해 "근처 와이파이"를 추천하기 위해 사용합니다. 위치 정보는 저장된 와이파이 항목과 함께 기기에만 저장되며, 외부로 전송되지 않습니다.

저장된 와이파이 정보

사용자가 저장·연결·공유한 와이파이 이름(SSID)과 비밀번호는 기기의 로컬 저장소(UserDefaults)에 저장됩니다. 이 정보는 사용자의 기기를 떠나지 않습니다.

데이터 공유

WifiSnap은 제3자와 데이터를 공유하지 않습니다. 와이파이 QR 코드는 사용자가 직접 공유 버튼을 눌러 전달할 때만, 사용자가 선택한 상대에게 전달됩니다.

데이터 삭제

저장된 와이파이는 목록에서 왼쪽으로 밀거나 길게 눌러 언제든 삭제할 수 있습니다. 앱을 삭제하면 기기에 저장된 모든 관련 데이터도 함께 제거됩니다.
