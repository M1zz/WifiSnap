# WifiSnap 설치 가이드

두 가지를 해주는 iOS 앱입니다.

1. **내 와이파이 공유**: 지금 연결된 와이파이를 감지해서, 친구가 카메라로 비추기만 하면 바로 연결되는 QR을 만들어줍니다.
2. **촬영 → 연결**: 와이파이 안내판(ID/PW)을 카메라로 찍으면 자동 인식해서 바로 연결해줍니다.

## 준비물

Mac + Xcode 15 이상, 애플 개발자 계정, 그리고 실제 아이폰이 필요합니다. 와이파이 연결(NEHotspotConfiguration)은 시뮬레이터에서 동작하지 않으므로 반드시 실기기로 테스트해야 합니다.

## 1. 프로젝트 만들기

1. Xcode → **File → New → Project → iOS → App**
2. Product Name: `WifiSnap`, Interface: **SwiftUI**, Language: **Swift**
3. Team은 본인 개발자 계정 선택, Bundle Identifier는 예: `com.본인아이디.wifisnap`
4. **Minimum Deployments를 iOS 17.0**으로 설정 (기본값이면 그대로 두어도 됩니다)

## 2. 소스 파일 추가

압축을 푼 `WifiSnap/WifiSnap/` 폴더 안의 파일들을 Xcode 프로젝트 내비게이터로 드래그해서 넣습니다 ("Copy items if needed" 체크).

- `WifiSnapApp.swift` — 기존 파일을 이 내용으로 교체
- `Views/ContentView.swift` — 기존 파일을 이 내용으로 교체
- `Views/ImagePicker.swift`, `Views/QRCodeSheet.swift`
- `Services/CurrentNetworkService.swift` — 현재 연결된 와이파이 SSID 감지
- `Services/TextRecognizer.swift` — Vision OCR (한국어+영어)
- `Services/WifiCredentialParser.swift` — ID/PW 파싱
- `Services/WifiConnector.swift` — 실제 와이파이 연결
- `Services/QRCodeGenerator.swift` — 공유용 WiFi QR 생성
- `Models/SavedNetwork.swift` — 저장된 네트워크 관리

`WifiSnap.entitlements`는 참고용입니다 — 3번 과정에서 capability를 추가하면 Xcode가 자동 생성해주므로 직접 넣지 않아도 됩니다.

## 3. Capability 두 개 추가 (가장 중요!)

1. 프로젝트 내비게이터에서 프로젝트 → 타깃 `WifiSnap` 선택
2. **Signing & Capabilities** 탭 → **+ Capability** 클릭
3. **Hotspot Configuration** 추가 — 와이파이 연결 기능에 필요. 없으면 연결 시도 시 앱이 즉시 크래시합니다.
4. 같은 방법으로 **Access Wi-Fi Information** 추가 — 현재 연결된 SSID 감지에 필요.

유료 개발자 계정에서는 둘 다 별도 승인 없이 바로 추가할 수 있습니다.

## 4. 권한 문구 추가

타깃 → **Info** 탭 → Custom iOS Target Properties에서 우클릭 → Add Row:

| Key | Value |
|---|---|
| `Privacy - Camera Usage Description` | 와이파이 안내판을 촬영해 아이디와 비밀번호를 인식합니다. |
| `Privacy - Location When In Use Usage Description` | 현재 연결된 와이파이 이름을 확인하기 위해 필요합니다. |

위치 권한이 필요한 이유: iOS는 주변 와이파이 이름으로 위치를 추정할 수 있다고 보기 때문에, SSID를 읽으려면 위치 권한을 요구합니다. 실제로 위치를 추적하지는 않습니다.

## 5. 실행

아이폰을 Mac에 연결(또는 무선 디버깅)하고 기기를 선택한 뒤 ⌘R로 실행합니다. 처음 실행 시 아이폰의 설정 → 일반 → VPN 및 기기 관리에서 개발자 앱을 신뢰해야 할 수 있습니다.

## 사용 방법

**내 와이파이 공유 (친구 연결용)**: 앱을 열면 맨 위에 지금 연결된 와이파이 이름이 자동으로 표시됩니다. 비밀번호를 한 번만 입력하고 "친구에게 보여줄 QR 만들기"를 누르면 표준 WiFi QR(`WIFI:S:...;T:WPA;P:...;;`)이 화면에 뜹니다. 친구가 아이폰이든 안드로이드든 기본 카메라로 비추면 탭 한 번으로 연결돼요. 한 번 만든 네트워크는 저장되어 다음부터는 비밀번호 입력 없이 바로 QR이 나옵니다. 비밀번호는 내 아이폰의 설정 → Wi-Fi → 연결된 네트워크의 ⓘ → 암호(Face ID 인증)에서 복사해올 수 있습니다.

**촬영 → 연결**: "안내판 촬영하기"로 ID/PW가 적힌 안내판을 찍으면 자동으로 인식됩니다. 인식 결과를 확인/수정한 뒤 "와이파이 연결"을 누르면 iOS 시스템 팝업("Wi-Fi 네트워크에 연결하시겠습니까?")이 뜨고, 허용하면 바로 연결됩니다. 이렇게 연결한 네트워크도 자동 저장되어 저장 목록의 QR 아이콘으로 언제든 공유할 수 있습니다.

## 알아두면 좋은 점

- iOS 보안 정책상 **어떤 앱도 이미 저장된 와이파이의 비밀번호를 읽을 수 없습니다.** 그래서 공유 기능은 "SSID는 자동 감지 + 비밀번호는 최초 1회 입력" 구조입니다. 이건 앱의 한계가 아니라 iOS의 설계입니다.
- 참고로 iOS 18부터는 기본 '암호(Passwords)' 앱 → Wi-Fi 섹션에서도 네트워크 QR 보기를 지원합니다. 앱 없이 급하게 공유할 때 유용합니다.
- 연결 시도는 앱이 포그라운드(화면에 떠 있는 상태)일 때만 가능합니다.
- WPA 비밀번호는 8자 이상이어야 API가 받아줍니다. 숫자만 있는 비밀번호도 문제없습니다.
- 저장소는 UserDefaults를 사용합니다. 보안이 중요하면 `SavedNetwork.swift`의 저장 로직을 Keychain으로 바꾸는 것을 권장합니다.
- 안내판 표기가 특이해서 인식이 빗나가도 텍스트 필드에서 바로 수정할 수 있게 해두었습니다.
